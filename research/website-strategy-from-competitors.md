# Website Strategy From Competitors

## Core Decision

Make `/miami-wedding-dj/` the primary Miami conversion hub for the wedding market, with `/miami/` acting as the broader market layer and other Miami service routes feeding into the same guided intake.

The competitive gap is not just prettier design. It is a better decision system for the buyer.

## Positioning TC Should Own

TC Audio Productions Miami should sit between three competitor categories:

- Luxury wedding production: polished visuals, calm confidence, elevated sound and lighting.
- Miami nightlife polish: energy, rhythm, bilingual music fluency, and dance-floor credibility.
- Corporate/event reliability: timelines, microphones, clean audio, venue coordination, backup mindset.

Recommended line:

guided planning. Human performance.

Do not frame this as automation. The trust signal is that the planning process organizes intake, music planning, pronunciation, timeline details, and vendor notes before the event team executes.

## Conversion Architecture

Competitors commonly say "Request a Quote." TC should say "Start Your Miami Event Plan."

The form should recommend a setup after enough detail is entered. This makes TC feel more premium because the site is actively helping the buyer make a decision.

Recommended package paths:

- Wedding + ceremony audio + bilingual MC: Bilingual Premium Wedding or Ceremony + Reception
- Wedding + ceremony audio: Ceremony + Reception
- Quinceanera/private family celebration: Quince / Private Celebration
- Corporate/hotel: Corporate / Hotel Event Production
- Lighting selected: Lighting / Production Support add-on
- Outdoor with no rain/cover note: custom review prompt

## Package Card Improvements

Package cards should be outcome-led, not only gear-led.

Must include:

- Best for label
- Primary outcome
- Included services
- Optional add-ons
- Quote-based language unless pricing is confirmed
- Recommendation tie-in from smart form

Recommended package set:

- Miami Reception DJ
- Ceremony + Reception
- Bilingual Premium Wedding
- Quince / Private Celebration
- Corporate / Hotel Event Production
- Lighting / Production Support

Highlight best-value:

Bilingual Premium Wedding for Miami multicultural weddings, because it combines the most differentiated concerns: ceremony, MC flow, bilingual announcements, timeline, and music balance.

## Proof Strategy

Do not fake social proof. Until real proof is approved, use process proof:

- Final event packet
- Timeline review
- Pronunciation guide
- Must-play/do-not-play fields
- Ceremony audio checklist
- Planner/venue coordination
- Backup signal path mindset
- Early arrival and clean setup language

When real proof is available, prioritize:

- Approved real event photos/video
- Own Google Business Profile reviews through compliant API/workflow
- The Knot/WeddingWire/Zola/Facebook links if TC has real profiles
- Planner or venue quotes only if permissioned and named accurately

## Spanish and Latin-First Strategy

Competitors either under-serve Spanish content or publish dense Spanish SEO pages. TC can win by using polished, Miami-appropriate Spanish with separate pages:

- `/es/dj-para-bodas-miami/`
- `/es/dj-bilingue-para-bodas-miami/`
- `/es/dj-para-quinceanera-miami/`
- `/es/blog/`

Spanish pages should be fully Spanish: nav, CTAs, form labels, FAQ labels, metadata, and structured content. Use `es-US`, not Spain-oriented phrasing.
